Calculator câmp

Calculează valorile pentru un câmp (atribut) pe baza unei formule specificate de dvs.

Expresii

Puteţi crea expresii pentru a calcula valori pentru câmpuri (atribute) noi sau existente.

După selectarea câmpului, se poate construi o expresie făcând clic în caseta pentru expresie pentru a lansa Builderul de expresii. În Builderul de expresii, se poate crea o expresie din orice combinaţie de alte câmpuri, operatori numerici, funcţii Python şi valori numerice sau şir. În cazul în care calculaţi valori pentru un câmp nou, trebuie să definiţi proprietăţile câmpului incluzând pseudonimul, tipul câmpului şi, pentru atribute text, lungimea textului conţinut în câmp.
