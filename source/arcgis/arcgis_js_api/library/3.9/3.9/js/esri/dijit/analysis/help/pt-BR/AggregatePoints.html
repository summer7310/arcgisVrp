Agregar Pontos

Esta ferramenta trabalha com uma camada de feições de ponto (a camada de análise) e uma camada de feições de área. Ela primeiro determina quais pontos caem dentro de cada área. Após determinar este relacionamento espacial do ponto na área, as estatísticas sobre todos os pontos na área são calculadas e atribuídas para a área. A estatística mais básica é a contagem do número de pontos dentro da área, mas você também pode obter outras estatísticas.

Por exemplo, suponha que você tenha feições de ponto dos locais das lojas de café e feições de área dos municípios, e você deseja resumir as vendas de café por município. Assumindo que as lojas de café têm um atributo TOTAL_SALES, você pode obter a soma de todos TOTAL_SALES dentro de cada município, ou o mínimo ou máximo TOTAL_SALES dentro de cada município, ou o desvio padrão de todas as vendas dentro de cada município.

O resultado é um mapa onde a cor de cada área é baseada no número total de pontos dividido pelo tamanho da área. A contagem de divisão por área é conhecida como normalização, e é uma técnica que os cartógrafos sempre utilizam para exibir os dados por áreas. A normalização ajuda a entender melhor os modelos espaciais, pois o tamanho da área é levado em conta quando áreas são visualmente comparadas umas com as outras.

Se Utilizar extensão de mapa atual estiver selecionada, somente as feições na camada da área e análise visíveis dentro da extensão de mapa atual serão analisadas. Se não estiver selecionada, todas as feições em ambas a camada de análise e a camada de área serão analisadas, mesmo que estiverem fora da extensão de mapa atual.

Escolher área

Selecione uma camada de área no mapa da web para a qual os pontos serão agregados. Se não existir nenhuma camada de limite no mapa da web, você poderá selecionar na lista de camadas de limite disponível para você no ArcGIS Online. Quaisquer pontos que caírem dentro dos limites de áreas dentro desta camada serão contados e opcionalmente resumidos utilizando suas opções de estatística.

Manter áreas sem pontos

Determina se as áreas que não têm nenhum ponto dentro delas aparecem na camada resultante. Se não selecionada, as áreas sem pontos não aparecerão no resultado. Se selecionada, as áreas sem pontos aparecerão no resultado.

Adicionar estatística

Você pode calcular estatísticas sobre atributos da camada de ponto de entrada como soma, média, mínimo, máximo e desvio padrão. A camada resultante conterá um novo atributo para cada estatística calculada. Qualquer número de estatística pode ser adicionado ao escolher um atributo e estatística (uma nova caixa suspensa aparecerá).

Escolher campo para agrupar por

Você pode criar grupos de statistica utilizando um atributo na camada de análise. Por exemplo, se você estiver agregando crimes para limites do bairro, você poderá ter um atributo Crime_type com cinco diferentes tipos de crimes. Cada um dos tipos de crime formam um grupo, e a estatística que você escolher será calculada para cada valor único de Crime_type. Quando você escolher um atributo do grupo, dois resultados serão criados: a camada resultante e uma tabela contendo a estatística para cada um dos grupos.

Nome da camada resultante

Este é o nome da camada que será criada em Meu Conteúdoe adicionada no mapa. O nome padrão é baseado no tipo de análise e nome da camada de análise. Se a camada já existir, você será avisado para confirmar se deseja substituí-la.

Utilizando a caixa suspensa Salvar resultado em, é possível especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo.
