Calcolatore campo

Consente di calcolare i valori di un campo (attributo) in base a una formula specificata.

Espressioni

È possibile creare espressioni per calcolare valori di campi (attributi) nuovi o esistenti.

Una volta selezionato il campo, è possibile creare un'espressione facendo clic sulla casella dell'espressione per avviare lo strumento Generatore di espressioni. Lo strumento Generatore di espressioni consente di creare un'espressione da qualsiasi combinazione di altri campi, operatori numerici, funzioni Python, valori numerici e valori di stringa. Per calcolare i valori di un nuovo campo, è necessario definire le proprietà del campo, inclusi alias e tipo di campo. Per gli attributi di testo, è necessario definire la lunghezza del testo che sarà contenuto nel campo.

Nome layer risultati

Questo è il nome del layer che verrà creato in I miei contenuti e aggiunto alla mappa. Il nome predefinito è basato sul nome del layer di analisi. Se il layer esiste già, verrà chiesto se si desidera sovrascriverlo.

Utilizzare la casella a discesa Salva risultato in per specificare il nome di una cartella di I miei contenuti in cui salvare il risultato.
