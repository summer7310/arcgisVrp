Lauka kalkulators

Kalkulē lauka (atribūta) vērtības, pamatojoties uz jūsu norādīto formulu.

Izteiksmes

Jūs varat izveidot izteiksmes, lai kalkulētu vērtības jauniem vai eksistējošiem laukiem (atribūtiem).

Kad izvēlas lauku, izteiksmi var veidot, noklikšķinot izteiksmes lodziņā, lai uzsāktu Izteiksmju veidotāju. Iekš Izteiksmju veidotāja, izteiksmi var veidot no citu lauku, skaitlisko operatoru, Python funkciju, skaitļu un virkņu vērtību kombinācijas. Ja jūs kalkulējat vērtības jaunam laukam, jums jādefinē lauka īpašības, iekļaujot aizstājvārdu, lauka tipu un teksta lauka garumu priekš teksta atribūta.

Rezultāta slāņa nosaukums

Šis ir slāņa nosaukums, kas tiks izveidots iekš jūsu Mans Saturs un pievienots kartei. Noklusējuma nosaukums ir balstīts uz analīzes slāņa nosaukumu. Ja slānis jau pastāv, jums tiks lūgts apstiprināt, vai vēlaties to pārrakstīt.

Lietojot Saglabāt rezultātu izvelkamajā lodziņā jūs varat norādīt nosaukumu mapē iekš Mans Saturs kur rezultāts tiks saglabāts.
