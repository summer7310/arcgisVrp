Resumir Dentro

Esta ferramenta localiza feições (e porções de feições) que estão dentro dos limites das áreas na camada de análise. Por exemplo:

  • Fornecida uma camada de limites da bacia e uma camada de limites de uso do solo por tipo de uso do solo, calcular a área total em acres do tipo de uso do solo para cada bacia.
  • Fornecida uma camada de lotes em um município e uma camada de limites da cidade, resumir o valor médio de lotes desocupados dentro de cada limite da cidade.

Se Utilizar extensão de mapa atual estiver selecionada, somente aquelas feições na camada de análise e camada a ser resumida que são visíveis dentro da extensão de mapa atual serão analisadas. Se não selecionada, todas as feições em ambas a camada de análise e a camada a ser resumida serão analisadas, mesmo que estiverem fora da extensão de mapa atual.

Escolher camada para resumir

As feições nesta camada que caem dentro dos limites das feições na camada de análise serão resumidas.

Adicionar estatística de

Você pode opcionalmente calcular estatísticas para atributos numéricos dos pontos que caem dentro dos limites das feições.

A caixa de seleção Contagem de pontos se selecionada, calculará o número total de pontos dentro de cada limite de área. Além disso, você pode calcular estatísticas para atributos numéricos dos pontos que estão dentro de cada limite de área.

  • Soma—O valor total do campo especificado.
  • Média—Calcula a média do campo especificado.
  • Mínimo—O menor valor no campo especificado.
  • Máximo—O maior valor no campo especificado.
  • Desvio Padrão—O desvio padrão dos valores no campo especificado.

Adicionar estatística de

Você pode opcionalmente calcular estatísticas para atributos numéricos das linhas mais próximas.

A caixa de seleção Comprimento total das linhas em se selecionada, calculará o comprimento total das linhas dentro de cada limite de área. Além disso, você pode calcular estatísticas para atributos numéricos das linhas que estão dentro de cada limite de área.

  • Soma—O valor total do campo especificado.
  • Média—Calcula a média do campo especificado.
  • Mínimo—O menor valor no campo especificado.
  • Máximo—O maior valor no campo especificado.
  • Desvio Padrão—O desvio padrão dos valores no campo especificado.

Adicionar estatística de

Você pode opcionalmente calcular estatísticas para atributos numéricos das áreas mais próximas.

A caixa de seleção Somar área utilizando se selecionada, calculará a área total das feições que estão dentro de cada limite de área da camada de análise. Além disso, você pode calcular estatísticas para atributos numéricos das feições que estão dentro de cada limite de área.

  • Soma—O valor total do campo especificado.
  • Média—Calcula a média do campo especificado.
  • Mínimo—O menor valor no campo especificado.
  • Máximo—O maior valor no campo especificado.
  • Desvio Padrão—O desvio padrão dos valores no campo especificado.

Escolher campo para agrupar por

Este é um atributo de Resumir feições que você pode utilizar para calcular estatísticas separadamente para cada valor de atributo único. Por exemplo, suponha que a camada de análise contém limites da cidade e as feições para Resumir são lotes. Um dos atributos dos lotes é Status que contém dois valores: VACANT e OCCUPIED. Para calcular a área total dos lotes ocupados e desocupados dentro dos limites das cidades, utilize Status como o grupo por atributo.

Nome da camada resultante

Este é o nome da camada que será criada em Meu Conteúdoe adicionada no mapa. O nome padrão é baseado no nome da camada de análise. Se a camada já existir, você será avisado para confirmar se deseja substituí-la.

Utilizando a caixa suspensa Salvar resultado em, é possível especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo.
