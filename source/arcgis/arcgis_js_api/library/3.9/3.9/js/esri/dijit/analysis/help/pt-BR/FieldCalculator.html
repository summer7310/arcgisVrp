Você pode criar expressões para calcular valores para novos ou campos existentes (atributos).

Após selecionar o campo, uma expressão pode ser construída ao clicar na caixa de expressão para iniciar o Construtor de Expressão. No Construtor de Expressão, uma expressão pode ser criada de qualquer combinação de outros campos, operadores numéricos, funções de Python e valores numéricos e caracteres. Se estiver calculando valores para um novo campo, você deverá definir as propriedades do campo inclusive nome alternativo, tipo de campo e atributos de texto, que terá o campo de comprimento do texto.

Nome da camada resultante

Este é o nome da camada que será criada em Meu Conteúdo e adicionada no mapa. O nome padrão é baseado no nome da camada de análise. Se a camada já existir, você será avisado para confirmar se deseja substituí-la.

Utilizando a caixa suspensa Salvar resultado em, é possível especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo.
