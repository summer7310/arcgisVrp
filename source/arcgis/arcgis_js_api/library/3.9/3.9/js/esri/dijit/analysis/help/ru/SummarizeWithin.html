Суммировать в пределах

Этот инструмент находит объекты (и части объектов), которые расположены в границах областей слоя анализа. Например:

  • Используя слой границ водоразделов и слой с границами типов землепользования, можно определить общую площадь земель различных типов для каждого водораздела.
  • Используя слой земельных участков в округе и слой с границами городов, можно вычислить среднюю стоимость свободных участков в границах городов.

Если опция Использовать текущий экстент карты (Use current map extent) включена, будут анализироваться только те объекты слоя анализа и слоя объединения, которые видны в момент анализа в текущем экстенте. Если опция отключена, будут анализироваться все объекты слоя анализа и слоя объединения, даже если они находятся вне текущего экстента карты.

Выберите слой для суммирования

Объекты этого слоя, которые расположены в границах объектов слоя анализа, будут суммированы.

Добавить статистику из

Дополнительно вы можете вычислить статистику числовых атрибутов точек, которые укладываются в пределы границ объектов.

При включении опции Число точек (Count of points) будет вычислено общее число точек в границах каждого объекта. Кроме того, вы можете вычислить статистику числовых атрибутов точек, которые укладываются в границы областей.

  • Сумма (Sum) – суммарное значение указанного поля.
  • Среднее (Average) – вычисляет среднее арифметическое указанного поля.
  • Минимум (Minimum) – определяет минимальное значение указанного поля.
  • Максимум (Maximum) – определяет максимальное значение указанного поля.
  • Стандартное отклонение (Std. Deviation) – стандартное отклонение для указанного поля.

Добавить статистику из

Дополнительно вы можете вычислить статистику числовых атрибутов ближайших линий.

При включении опции Общая длина линий в (Total length of lines in) будет вычислена общая длина линий в границах каждой области. Кроме того, вы можете вычислить статистику числовых атрибутов линий, которые находятся в границы областей.

  • Сумма (Sum) – суммарное значение указанного поля.
  • Среднее (Average) – вычисляет среднее арифметическое указанного поля.
  • Минимум (Minimum) – определяет минимальное значение указанного поля.
  • Максимум (Maximum) – определяет максимальное значение указанного поля.
  • Стандартное отклонение (Std. Deviation) – стандартное отклонение для указанного поля.

Дополнительно вы можете вычислить статистику числовых атрибутов ближайших областей.

При включении опции Применение суммирования площади (Sum area using) будет вычислена общая площадь объектов в границах каждой области слоя анализа. Кроме того, вы можете вычислить статистику числовых атрибутов объектов, которые находятся в границы областей.

  • Сумма (Sum) – суммарное значение указанного поля.

  • Максимум (Maximum) – определяет максимальное значение указанного поля.
  • Стандартное отклонение (Std. Deviation) – стандартное отклонение для указанного поля.

Выбрать поле для группировки

Это атрибут объектов Суммирования (Summarize), который может использоваться для вычисления статистики отдельно для каждого уникального значения атрибута. Например, предположим слой содержит границы городов и объектами Суммирования (Summarize) являются земельные участки. Один из атрибутов земельных участков – это статус, Status, который содержит два значения: VACANT и OCCUPIED (Свободен и Занят). Для вычисления общей площади, а также свободных и занятых участков в границах городов, используйте Status для группировки по атрибуту.

Имя слоя результата

Это имя слоя, который будет добавлен в Мои ресурсы (My Contents) и на карту. Имя слоя по умолчанию базируется на имени слоя анализа. Если такой слой уже существует, появится запрос на его перезапись.

С помощью раскрывающегося поля Сохранить результат в (Save result in) вы можете задать имя папки в разделе Мои ресурсы (My Contents), в которую будет записан результат.
