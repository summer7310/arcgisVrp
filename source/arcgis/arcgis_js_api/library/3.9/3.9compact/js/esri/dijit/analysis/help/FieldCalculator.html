Field Calculator

Calculates values for a field (attribute) based on a formula you specify.

Expressions

You can create expressions to calculate values for new or existing fields (attributes).

After selecting field, an expression can be built by clicking in the expression box to launch the Expression Builder. In the Expression Builder, an expression can be created from any combination of other fields, numerical operators, Python functions, and numeric and string values. If you are calculating values for a new field, you must define the field's properties including alias, field type, and for text attributes, the length of text the field will contain.

Result layer name

This is the name of the layer that will be created in My Contents and added to the map. The default name is based on the analysis layer name. If the layer already exists, you will be asked to confirm if you want to overwrite it.

Using the Save result in drop-down box, you can specify the name of a folder in My Contents where the result will be saved.
