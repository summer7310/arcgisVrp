Aggrega punti

Questo strumento viene utilizzato con un layer di feature punto (layer di analisi) e un layer di feature area e in primo luogo determina i punti compresi in ciascuna area. Dopo aver stabilito questa relazione spaziale tra i punti dell'area, le statistiche relative a tutti i punti compresi nell'area vengono calcolate e assegnate all'area. La statistica più semplice è costituita dal numero di punti compresi nell'area, tuttavia è possibile ottenere anche altre statistiche.

Si supponga, ad esempio, che siano presenti feature punto relative alla posizione dei bar e feature area per le regioni e che si desideri riepilogare le vendite di caffè per regione. Presupponendo che ai bar sia associato un attributo TOTAL_SALES, è possibile ottenere la somma dei valori di tutti gli attributi TOTAL_SALES in ogni regione, il valore minimo o massimo di TOTAL_SALES in ogni regione oppure la deviazione standard di tutte le vendite in ogni regione.

Nella mappa risultante, il colore di ogni area è basato sul numero totale di punti diviso per le dimensioni dell'area. La divisione del numero di punti per l'area è detta normalizzazione ed è una tecnica utilizzata dai cartografi per visualizzare i dati in base alle aree. La normalizzazione consente di comprendere meglio i modelli spaziali, perché vengono prese in considerazione le dimensioni dell'area durante il confronto visivo delle aree.

Se si seleziona Usa estensione mappa corrente, verranno analizzate solo le feature nel layer di analisi e di area visibili nell'estensione mappa corrente. Se questa opzione è deselezionata, verranno analizzate tutte le feature sia nel layer di analisi che in quello di area, anche se sono esterne all'estensione mappa corrente.

Scegliere l\'area

Selezionare un layer di area della mappa Web a cui verranno aggregati i punti. Se la mappa Web non include layer dei confini, è possibile sceglierne uno nell'elenco dei layer dei confini disponibili in ArcGIS Online. Tutti i punti compresi nei confini delle aree in questo layer verranno conteggiati e, se si desidera, inclusi nel riepilogo in base alle statistiche selezionate.

Mantieni le aree senza punti

Determina se nel layer dei risultati saranno incluse anche le aree senza punti. Se l'opzione non è selezionata, le aree senza punti non verranno incluse nei risultati. Se l'opzione è selezionata, le aree senza punti verranno incluse nei risultati.

Aggiungi statistiche

È possibile calcolare le statistiche relative agli attributi del layer di punti di input, ad esempio la somma, la media, il valore minimo, il valore massimo e la deviazione standard. Il layer dei risultati includerà un nuovo attributo per ogni statistica calcolata. È possibile aggiungere un numero qualsiasi di statistiche scegliendo un attributo e una statistica (apparirà una nuova casella a discesa).

Scegli campo in base a cui raggruppare

È possibile creare gruppi statistici utilizzando un attributo nel layer di analisi. Ad esempio, se si stanno aggregando i reati ai confini della vicinanza, è possibile avere un attributo Tipo_reato con cinque diversi tipi di reato. Ciascun tipo di reato forma un gruppo e le statistiche selezionate saranno calcolate per ogni valore univoco di Tipo_reato. Quando si sceglie un attributo di raggruppamento, si creano due risultati: il layer risultati e una tabella che contiene le statistiche per ogni singolo gruppo.

Nome layer risultati

Questo è il nome del layer che verrà creato in I miei contenuti e aggiunto alla mappa. Il nome predefinito è basato sul tipo di analisi e sul nome del layer di analisi. Se il layer esiste già, verrà chiesto se si desidera sovrascriverlo.

Utilizzare la casella a discesa Salva risultato in per specificare il nome di una cartella di I miei contenuti in cui salvare il risultato.
