Resumir cerca de

Esta herramienta busca entidades que están dentro de una distancia especificada de las entidades de la capa de análisis. La distancia se puede medir como distancia en línea recta, como distancia de tiempo de conducción (por ejemplo, en 10 minutos) o como distancia de conducción (a 5 kilómetros). A continuación, se calculan las estadísticas para las entidades cercanas. Por ejemplo:

  • Calcula la población total que se encuentra a menos de cinco minutos de tiempo de conducción de la ubicación propuesta para una nueva tienda.
  • Calcula el número de incorporaciones a la autovía a una distancia de conducción de una milla de la ubicación propuesta para una nueva tienda con el fin de usarlas como medida de la accesibilidad de una tienda.

Si se ha activado Utilizar la extensión de mapa actual al buscar ubicaciones, solo se analizarán las entidades de la capa de análisis y la capa para resumir que estén visibles en la extensión de mapa actual. Si no se ha activado, se analizarán todas las entidades de la capa de análisis y de la capa para resumir, aunque estén fuera de la extensión de mapa actual.

Find nearest features using a

Puedes definir la clase de medición de distancia que deseas utilizar: la distancia en línea recta, la distancia de conducción o bien el tiempo de conducción. La distancia y el tiempo de conducción utilizan la red de carreteras y respetan restricciones como las vías de sentido único. El tiempo de conducción respeta las velocidades de tráfico seleccionadas: las basadas en el tráfico típico o en el tráfico en directo. Las velocidades del tráfico en directo se recopilan mediante las lecturas actuales de los sensores, mientras que las velocidades del tráfico típico se calculan a partir de las medias históricas.

Puedes introducir varias distancias. Se resumirán las entidades que estén dentro (o equivalgan a) las distancias introducidas.

Este es un atributo de las entidades próximas que puedes utilizar para calcular las estadísticas individuales de cada valor de atributo exclusivo. Por ejemplo, supongamos que tu capa de análisis contiene parcelas vacantes en venta y que tus entidades próximas son huellas de edificios. Uno de los atributos de las huellas de edificios es UseType que contiene valores como RETAIL, RESIDENTIAL, MANUFACTURING, PUBLIC, etc. Para caracterizar la mezcla de usos de edificios junto a cada parcela vacante, puedes calcular el área total de cada tipo exclusivo de uso de edificio en un área de dos kilómetros de las parcelas vacantes, utilizando UseType como el grupo por atributo.

Result layer name

Este es el nombre de la capa que se creará en Mi contenido y se agregará al mapa. El nombre predeterminado se basa en el nombre de la capa de análisis. Si la capa ya existe, se te pedirá que confirmes si deseas sobrescribirla.

En el cuadro desplegable Guardar resultado en, puedes especificar el nombre de una carpeta de Mi contenido en la que se guardará el resultado.
