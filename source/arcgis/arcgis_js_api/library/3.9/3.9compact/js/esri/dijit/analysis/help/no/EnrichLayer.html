Suppleringsdata

Med dette verktøyet supplerer du dataene dine ved å innhente fakta om personer, steder og virksomheter som befinner seg rundt datalokasjonene, og du kan i tillegg få svar på spørsmål om lokasjonene som du ikke kan få svar på med kart alene. For eksempel: Hva slags mennesker bor her? Hva liker menneskene som bor i dette området, å gjøre? Hvilke vaner og hvilken livsstil har de? Hvilken type virksomheter finnes i dette området?

Resultatet blir et nytt lag som inneholder all demografisk og geografisk informasjon fra gitte datasamlinger. Den nye informasjonen legges til som felter i tabellen.

Hvis det er merket av for Use current map extent, analyseres kun geoobjektene i analyselaget og områdelaget som vises i gjeldende kartutstrekning. Hvis det ikke er merket av for dette alternativet, analyseres alle geoobjekter i både analyselaget og områdelaget, selv om de befinner seg utenfor gjeldende kartutstrekning.

Vis tilgjengelige data for

Velg området ditt, og få en liste over tilgjengelige datasamlinger og attributter innenfor datasamlingen. Du kan velge en hel samling eller utvide samlingen og velge ett eller flere attributter fra samlingen. De valgte datasamlingene og attributtene legges til i resultatlaget.

Definer områdene du vil supplere

Du kan forbedre disse områdene ved hjelp av tilleggsinformasjon som finnes i datasamlingen. Hvis analyselaget inneholder punkter eller linjer, opprettes det områder rundt hvert punkt eller hver linje ved hjelp av luftlinjeavstand eller kjøretid.

Navn på resultatlag

Dette er navnet på laget som opprettes i Mitt innhold og legges til kartet. Standardnavnet er basert på navnet på analyselaget. Hvis laget finnes allerede, blir du bedt om å bekrefte at du vil overskrive det.

Bruk rullegardinlisten Save result in til å angi navnet på en mappe i Mitt innhold som resultatet skal lagres i.
