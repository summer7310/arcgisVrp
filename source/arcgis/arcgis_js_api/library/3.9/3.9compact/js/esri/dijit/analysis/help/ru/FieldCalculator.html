Калькулятор поля

Вычисляет значения для поля (атрибута) согласно заданной формуле.

Выражения

Вы можете создать выражения надписи для вычисления значений новых или имеющихся полей (атрибутов).

После выбора поля можно создать выражение, щелкнув в поле выражения, чтобы открыть Конструктор выражений (Expression Builder). В Конструкторе выражений (Expression Builder) можно создать формулу, используя комбинации любых полей, числовые операторы, функции Python и числовые или строковые значения. Если вы вычисляете значения нового поля, необходимо задать свойства поля, включая псевдоним, тип поля, а для текстовых атрибутов – длину текстового поля.

Имя слоя результата

Это имя слоя, который будет добавлен в Мои ресурсы (My Contents) и на карту. Имя слоя по умолчанию базируется на имени слоя анализа. Если такой слой уже существует, появится запрос на его перезапись.

С помощью раскрывающегося поля Сохранить результат в (Save result in) вы можете задать имя папки в разделе Мои ресурсы (My Contents), в которую будет записан результат.
