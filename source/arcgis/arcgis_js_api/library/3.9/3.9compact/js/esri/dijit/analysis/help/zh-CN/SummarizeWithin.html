范围内汇总

此工具用于查找分析图层中处于区域边界内的要素（和部分要素）。例如：

  • 按土地使用类型给定流域边界图层和土地使用边界图层，计算每个流域的土地使用类型的总面积。
  • 给定县内宗地的图层和城市边界图层，汇总各城市边界内闲置宗地的平均值。

如果选中 使用当前地图范围，则仅分析当前地图范围内可见的分析图层和待汇总图层中的要素。如果未选中，则会对分析图层和待汇总图层内的所有要素进行分析，包括超出当前地图范围的要素。

选择汇总图层

汇总此图层中位于分析图层的要素边界内的要素。

添加统计数据

对于要素边界内的点，您可以计算其数值属性的统计数据。

如果选中 点计数复选框，则会计算每个区域边界内点的总数。另外，您还可以计算每个区域边界内点的数值属性的统计数据。

  • 总和 - 指定字段的合计值。
  • 平均值 - 指定字段的平均值。
  • 最小值 - 指定字段的最小值。
  • 最大值 - 指定字段的最大值。
  • 标准差 - 指定字段中值的标准差。

添加统计数据

您还可以计算邻近线的数值属性的统计数据。

如果选中 线的总长度复选框，则会计算每个区域边界内线的总长度。另外，您还可以计算每个区域边界内线的数值属性的统计数据。

  • 总和 - 指定字段的合计值。
  • 平均值 - 指定字段的平均值。
  • 最小值 - 指定字段的最小值。
  • 最大值 - 指定字段的最大值。
  • 标准差 - 指定字段中值的标准差。

添加统计数据

您还可以计算邻近区域的数值属性的统计数据。

如果选中 计算总面积复选框，则会计算分析图层中每个区域边界内要素的总面积。另外，您还可以计算每个区域边界内要素的数值属性的统计数据。

  • 总和 - 指定字段的合计值。
  • 平均值 - 指定字段的平均值。
  • 最小值 - 指定字段的最小值。
  • 最大值 - 指定字段的最大值。
  • 标准差 - 指定字段中值的标准差。

选择作为分组依据的字段

这是 汇总要素的一个属性，可用于分别计算每个唯一属性值的统计数据。例如，假设分析图层包含城市边界， 汇总要素为宗地。 Status 是宗地的一个属性，它包含两个值：VACANT 和 OCCUPIED。要计算城市边界内空闲和占用宗地的总面积，请使用 Status 作为按属性分组的条件。

结果图层名称

在 我的内容中创建并添加到地图中的图层名称。默认名称参照分析图层的名称。如果图层已经存在，则系统会要求您确认是否要覆盖它。

通过使用 将结果保存到下拉框，可以在 我的内容中指定文件夹的名称，结果将保存到该文件夹中。
