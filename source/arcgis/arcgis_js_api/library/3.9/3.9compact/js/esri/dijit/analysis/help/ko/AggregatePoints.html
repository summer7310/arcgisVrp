포인트 집계

이 도구는 포인트 피처 레이어(분석 레이어) 및 영역 피처 레이어에서 작동합니다.먼저 각 영역에 속한 포인트를 확인합니다.이 영역 내 포인트 공간 관계를 확인한 후에는 영역 내 모든 포인트에 대한 통계가 계산되고 영역에 할당됩니다.가장 기본적인 통계는 영역 내에 있는 포인트 수이지만 다른 통계도 확인할 수 있습니다.

예를 들어 커피숍 위치에 대한 포인트 피처와 국가에 대한 영역 피처가 있다고 가정하고 도시별 커피 판매량을 요약해 보겠습니다.커피숍에 TOTAL_SALES 속성이 있는 것으로 가정할 때 각 도시 내의 모든 TOTAL_SALES 합계, 각 도시 내의 최소 또는 최대 TOTAL_SALES 또는 각 도시 내의 모든 판매량에 대한 표준 편차를 구할 수 있습니다.

영역 크기로 구분된 총 포인트 수를 기반으로 각 영역의 색상이 지정된 맵이 생성됩니다.영역별로 개수를 나누는 것을 정규화라고 하며, 이는 지도 제작자가 데이터를 영역별로 표시하는 데 늘 사용하는 기법입니다.정규화를 사용하면 영역을 시각적으로 서로 비교할 때 영역의 크기가 고려되므로 공간 패턴을 이해하는 데 도움이 됩니다.

현재 맵 범위 사용을 선택한 경우 현재 맵 범위 내에 표시된 분석 레이어와 영역 레이어의 피처만 분석되고,이 옵션의 선택을 취소한 경우 현재 맵 범위 외부에 있는 피처를 포함하여 분석 레이어와 영역 레이어에 있는 모든 피처가 분석됩니다.

영역 선택

웹 맵에서 포인트를 집계할 영역 레이어를 선택합니다.웹 맵에 경계 레이어가 없는 경우 ArcGIS Online에서 사용할 수 있는 경계 레이어 목록에서 선택할 수 있습니다.이 레이어의 영역 경계 내에 있는 모든 포인트 수가 계산되고 사용자가 선택한 통계를 사용하여 선택적으로 요약됩니다.

포인트 없는 영역 유지

내부에 포인트가 없는 영역을 결과 레이어에 표시할지 여부를 결정합니다.이 옵션을 선택하지 않으면 포인트가 없는 영역이 결과에 표시되지 않고,이 옵션을 선택하면 포인트가 없는 영역이 결과에 표시됩니다.

통계 추가

입력 포인트 레이어의 속성에 대한 통계(예: 합계, 평균, 최소값, 최대값, 표준 편차)를 계산할 수 있습니다.결과 레이어에는 계산된 각 통계에 대한 새 속성이 포함됩니다.속성과 통계를 선택(새 드롭다운 상자가 나타남)하여 원하는 개수의 통계를 추가할 수 있습니다.

그룹화할 필드 선택

분석 레이어의 속성을 사용하여 통계 그룹을 생성할 수 있습니다.예를 들어 네이버후드 경계 지역의 범죄를 집계하는 경우 5가지 범죄 유형으로 구성된 Crime_type 속성을 사용할 수 있습니다.각 고유한 범죄 유형이 그룹을 구성하며 Crime_type의 각 고유한 값에 대해 선택한 통계가 계산됩니다.그룹화 속성을 선택하면 두 가지 결과, 즉결과 레이어와 각 고유 그룹에 대한 통계가 포함된 테이블이 생성됩니다.

결과 레이어 이름

내 컨텐츠에서 생성되어 맵에 추가되는 레이어의 이름입니다.기본 이름은 분석 유형과 분석 레이어 이름을 기반으로 합니다.레이어가 이미 있는 경우 덮어쓸지 확인하는 메시지가 나타납니다.

결과 저장: 드롭다운 상자를 사용하여 내 컨텐츠에서 결과를 저장할 폴더의 이름을 지정할 수 있습니다.
